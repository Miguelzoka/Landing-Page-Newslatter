# 📈 Marketing Landing Page

Landing Page desenvolvida utilizando **HTML5** e **CSS3**, com foco na divulgação de serviços de marketing digital e conversão de visitantes em clientes.

## 🚀 Tecnologias

* HTML5
* CSS3

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar a criação de landing pages modernas, explorando conceitos de responsividade, hierarquia visual, experiência do usuário (UX) e técnicas de design voltadas para marketing digital.
